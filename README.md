# Getting Started
A maze generating algorithim visualized with pygame
### Prerequisites
* python
  ```sh
  python
  ```
* pygame
  ```sh
  python -m pip install pygame
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mKupiec2/maze-generation.git
   ```
2. Run the visualizer
   ```sh
   python maze.py
   ```
